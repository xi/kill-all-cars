# Kill All Cars

This is a game (in early development) where you are a city planner tasked with
improving the living conditions in the city. It takes a lot of
inspiration from games like [freeways](https://captaingames.itch.io/freeways)
or [SimCity](https://en.wikipedia.org/wiki/Simcity). Just as in real-life, the
best way to improve cities is usually to remove car infrastructure.
